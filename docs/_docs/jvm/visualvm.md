# VisualVM

> java8之后 jdk不再附带 VisualVM \
> [下载页面](http://visualvm.github.io) \
> [github](https://github.com/oracle/visualvm)

## 运行参数配置文件 

> `visualvm/etc/visualvm.conf`\
> 例如配置`jdkhome`:`visualvm_jdkhome="C:\Program Files\Java\jdk-17.0.2"`

## 运行文件

> `visualvm/bin/visualvm`
