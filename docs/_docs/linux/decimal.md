## 整数 十进制 转 二进制

> 123 = 128 - 5 = 128 - (4 + 1)
> > 2^7 - (2^2 + 1) \
> > 1000 0000 - (100 + 1) \
> > 1000 0000 - 101 \
> > 111 1011
>
> 123 = 64 + 32 + 16 + 8 + 2 + 1
> > 1 x2^6 + 1 x2^5 + 1 x2^4 + 1 x2^3 + 0 x2^2 + 1 x2^1 + 1 x2^0
> > 111 1011
>
> **短除法** 计算到商为0 ,余数逆序 即为 二进制
> > 123 / 2 = 61 余 1 \
> > 61 / 2 = 30  余 1 \
> > 30 / 2 = 15  余 0 \
> > 15 / 2 = 7   余 1 \
> > 7 / 2 = 3    余 1 \
> > 3 / 2 = 1    余 1 \
> > 1 / 2 = 0    余 1 

## 小数 十进制 转 二进制

> 小数位乘 2 取整数位；剩余小数位继续乘 2 取整数位；直到小数位为 0 \
> > 0.125 * 2 = 0.25 \
> > 0.25 * 2 = 0.5 \
> > 0.5 * 2 = 1.0
